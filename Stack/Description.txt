Реализуйте следующие операции стека:

void init();
Инициализация стека.

void push(int x);
Добавить элемент в стек.

int pop();
Удалить топовий элемент и вернуть его значение.
Если стек пустой - вернуть минимальное значение int.

int top();
Вернуть значение топового элемента.
Если стек пустой - вернуть минимальное значение int.

int size();
Вернуть размер стека.

Суммарное число вызовов до 100000.
