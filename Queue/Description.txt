Реализуйте следующие операции очереди:

void init();
Инициализация очереди.

void push(int x);
Добавить элемент в очередь.

int pop();
Удалить первый элемент и вернуть его значение.
Если очередь пустая - вернуть минимальное значение int.

int front();
Вернуть значение первого элемента.
Если очередь пустая - вернуть минимальное значение int.

int size();
Вернуть размер очереди.

Суммарное число вызовов до 100000.
