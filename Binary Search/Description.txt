Даны числа отсортированые по возрастанию.

const int *reinvent_lower_bound(const int *first, const int *last, int value);
Возвращает указатель на самый левый элемент в диапазаное [first, last), который больше или равен искомому, или указатель на last, если элемента нет.
ВНИМАНИЕ: last - указатель на элемент за пределами диапазона. Нельзя брать значение по этому указателю.
